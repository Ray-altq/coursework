# Клеточный автомат 

Интерактивный симулятор клеточного автомата **«Игра Жизни» Джона Конвея** на Java с графическим интерфейсом (Swing).  
Проект сделан как курсовая работа: демонстрирует алгоритмы, структуры данных и работу с GUI.

---

## Возможности

- **Моделирование Game of Life** по правилам **B/S** (например, `B3/S23`)
- **Разреженное “бесконечное” поле**: хранится только набор живых клеток (координаты)
- **Популярные правила** выбираются кнопками 
- **Редактирование поля мышью**:
  - ЛКМ — поставить клетку / паттерн
  - ПКМ — стереть клетку / паттерн
- **Готовые паттерны (Stamp)**: глайдеры, осцилляторы, корабли, пушка Госпера и др.
- **Random**: случайное заполнение области одиночными клетками и паттернами
- **Управление симуляцией**: Start/Pause, Step, Reset, Speed (шаги/сек)
- **Навигация по полю**:
  - панорамирование (стрелки, колёсико)
  - масштабирование (Ctrl/⌘ + колёсико)
- **Топология**:
  - `Torus` — замкнуто по X и Y (обёртка по модулю)

- **Java 17+** (желательно)

### Запуск из IDE
1. Открыть проект в IntelliJ IDEA / Eclipse
2. Запустить главный класс:
